# Processed analytical outputs

This directory contains processed analytical outputs generated by the stepwise notebooks and used directly in the manuscript results, figures, and appendices.

## Files

- `SPI_monthly.csv` – monthly Standardized Precipitation Index (SPI) series at 3-, 6-, and 12-month aggregation windows
- `SGI_monthly.csv` – monthly Standardized Groundwater level Index (SGI) series for all piezometers and aggregation windows
- `drought_metrics_cluster_summary.csv` – cluster-level drought-event metrics including duration, propagation probability, recovery, resistance, resilience, and vulnerability
- `DIPI_components.csv` – point-scale DIPI components, normalized subcomponents, and final DIPI values used in mapping and appendix analyses

These files represent the derived analytical products that directly support the manuscript interpretation and figure source-data packages.
