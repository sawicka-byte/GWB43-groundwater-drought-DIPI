# Processed analytical outputs

This directory contains derived analytical products generated by the reproducibility workflow and used in the manuscript results, figures, appendices, and quality-control analyses.

## Files

- `SPI_monthly.csv` — monthly Standardized Precipitation Index series at 3-, 6-, and 12-month aggregation windows
- `SGI_normal_score_long.csv` — well-level SGI-3, SGI-6, and SGI-12 series calculated with the revised calendar-month non-parametric normal-score procedure
- `group_SGI.csv` — aquifer-group mean SGI series with dynamic monitoring support and the minimum-well rule
- `drought_events_well.csv` — detected groundwater drought events for individual piezometers
- `drought_metrics_well.csv` — well-level drought metrics, including duration, severity, intensity, CRT, CRS, VUL, recovery, and propagation probability
- `drought_events_group.csv` — detected drought events for aquifer-group SGI series
- `drought_metrics_group.csv` — aquifer-group drought metrics and monitoring-support information
- `propagation_matches_well.csv` — matches between SGI-3 and SGI-12 drought events at well level
- `propagation_matches_group.csv` — matches between SGI-3 and SGI-12 drought events at aquifer-group level
- `DIPI_components.csv` — raster-sampled exposure, pressure, sensitivity, normalized subcomponents, weighted DIPI, equal-weight DIPI, and legacy-value quality-control fields
- `DIPI_aquifer_group_summary.csv` — descriptive summary of DIPI components and variants by aquifer group

## Key definitions

Groundwater drought months are defined by:

```text
SGI < -1.0
```

The main DIPI variant is:

```text
DIPI_weighted = 0.4 × exposure + 0.3 × pressure + 0.3 × sensitivity
```

The equal-weight robustness variant is:

```text
DIPI_equal_weight = (exposure + pressure + sensitivity) / 3
```

The DIPI maps presented in the manuscript were calculated from GIS rasters. The point-scale DIPI table contains values sampled at piezometer locations and should not be interpreted as a substitute for the full raster products.
