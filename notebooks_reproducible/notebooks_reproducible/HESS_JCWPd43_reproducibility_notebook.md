# HESS_JCWPd43_reproducibility_notebook

## Purpose
This master notebook reproduces the full workflow used in the manuscript from processed input datasets to final DIPI outputs and figure source tables.

## Full workflow
1. Load processed monthly groundwater and precipitation datasets
2. Calculate SPI and SGI
3. Detect drought events
4. Compute event metrics
5. Construct exposure, pressure, and sensitivity layers
6. Calculate final DIPI values
7. Export source tables for all manuscript figures
8. Export summary statistics for cluster comparison

- `data_processed/`

## Expected outputs

- summary statistics
