# Figure source data index

This directory contains source-data packages for all main-text figures and appendices used in the manuscript:

**Integrating propagation and recovery dynamics into groundwater drought vulnerability assessment through exposure, pressure, and aquifer system response**

## Main figures
- **fig01/** – study area map layers (piezometers, meteorological stations, GDEs, mining areas, rivers, lakes, GWB boundary)
- **fig02/** – conceptual hydrogeological cross-section note
- **fig03/** – conceptual synthesis diagram note
- **fig04/** – SPI–SGI time-series source data
- **fig05_06/** – cluster-level drought metrics used in multi-panel summary plots
- **fig07_08/** – point-based DIPI component values and final DIPI values

## Appendices
- **Appendix_A/** – point-scale DIPI subcomponents and normalized component layers
- **Appendix_B/** – SGI–SPI lagged correlation heatmap source matrix

Each subfolder contains the numerical tables or interpretative notes required to reproduce the corresponding figure panels.
Conceptual figures (Fig. 2 and Fig. 3) are accompanied by interpretative notes instead of standalone numerical datasets.
