# Figure 3 source data note

Figure 3 is a conceptual synthesis diagram illustrating differences in groundwater drought propagation, attenuation, delay, and system memory across the three aquifer groups.

The panel summarizes the process-based interpretation supported by the SGI–SPI lag analysis, event-based drought metrics, and cluster-level hydrogeological contrasts presented in the manuscript.

As an interpretative conceptual figure, it is not associated with a standalone numerical source-data table.
